> **파일**: `design/mockups/ja/07_booking_form.html`

### C-03 · 소요시간 표시바

선택된 슬롯과 총 소요시간을 조합해 예약 시작~종료 시간을 표시.

| 항목 | 값 |
|------|----|
| 타입 | Flex 바 (div `.duration-bar`) |
| 높이 | 자동 (패딩 8px) |
| 배경 | `white` |
| 테두리 | 상단 `.5px solid #E5E5E0` |

| # | 요소 | 값 |
|---|------|-----|
| C-03-1 | 레이블 | `選択時間`, 11px / `#888` |
| C-03-2 | 시작 시간 | `#dbTime`, 13px / 500 / `#8E44AD` |
| C-03-3 | 화살표 | `→`, 12px / `#bbb` |
| C-03-4 | 종료 시간 | `#dbEnd`, 12px / `#555` |
| C-03-5 | 소요시간 | `(N分)`, 11px / `#888` |

**계산**:
```
종료시간 = 시작시간(분) + totalDur
#dbEnd = floor(종료분/60) + ':' + (종료분%60).padStart(2,'0')
```

## D. JS 상태 관리

| 변수 | 타입 | 초기값 | 설명 |
|------|------|--------|------|
| `selectedTime` | string | `'13:00'` | 선택된 시작 시간 |
| `totalDur` | number | 150 | 총 소요 시간(분) |

| 함수 | 역할 |
|------|------|
| `buildAvailGrid()` | 가용 슬롯 그리드 전체 렌더링 |
| `selectSlot(time, si)` | 슬롯 선택, `selectedTime` 갱신, 종료 시간 계산, 그리드 재렌더링 |
| `calcDuration()` | 시술 소요시간 합산 → `totalDur` 갱신 → `selectSlot` 재호출 |
| `removeService(el)` | 시술 항목 제거 → `calcDuration()` 재호출 |
| `addService()` | 시술 추가 (랜덤 데모, 실제는 메뉴 선택 팝업) |
| `selectStaff(el)` | 담당 스태프 선택 |
| `searchCustomer(val)` | 고객 검색 (실제 구현 시 API 호출) |
| `clearCustomer()` | 선택된 고객 초기화 |
| `changeDate(dir)` | 날짜 이동 |
| `goSlotToday()` | 오늘로 이동 |
| `saveBooking()` | 유효성 검사 후 저장 실행 |

## E. 유효성 검사 규칙

| 필드 | 규칙 | 에러 메시지 |
|------|------|-------------|
| 고객 | 반드시 선택 | `顧客を選択してください` |
| 시술 메뉴 | 최소 1개 | `メニューを1つ以上選択してください` |
| 담당 스태프 | 반드시 선택, 휴무 불가 | `担当スタッフを選択してください` |
| 날짜·시간 | 반드시 선택, 과거 불가 | `日時を選択してください` |
| 스태프 가용 | 해당 시간 예약 없어야 함 | `このスタッフはすでに予約があります` |
