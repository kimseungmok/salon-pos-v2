# 08 · ウェイティング管理 — 화면 정의서

> **파일**: `design/mockups/v2/ja/08_waiting_list.html`
> **목적**: 워크인/대기 고객 큐 관리. 대기 등록 → 호출 → 확정(예약 전환)/취소까지의 흐름을 한 화면에서 처리.
> **진입**: 사이드바 → ウェイティング / 06_booking_calendar Zone F-3 「もっと見る」
> **이동**: 確定 클릭 → `07_booking_form` (좌석 배정·시술 시작) / 신규 추가 → 같은 화면 내 인라인 폼

## 4. Zone C — ウェイティングリスト

### 4-1. 카드 구성 (待機中 / 呼び出し済み)

```
┌──┬───────────────────────────────────────────────────┬──────────────┐
│▲▼│ 1  森 かれん          090-XXXX-1234                │ [呼び出す]    │
│  │    カット＋カラー・90分 / 指定: Yuki                │ [取消]        │
│  │    10:05 受付 ・ 19分待ち                           │              │
└──┴───────────────────────────────────────────────────┴──────────────┘
```

| 요소 | 설명 |
|------|------|
| 순번 배지 | 대기 순서 (1부터). 호출됨 탭에서는 숨김 |
| ▲▼ 재정렬 | 待機中 탭에서만 표시. 순서 변경 시 即시 리스트 재배열 |
| 고객명·전화번호 | 전화번호는 마스킹 표시 |
| 메뉴·소요시간·지정 스태프 | "指定なし"면 회색 표기 |
| 접수 시각·경과 시간 | `HH:MM 受付 ・ N分待ち` — 10분 미만 회색, 10~20분 황색, 20분 이상 적색 강조 |
| 액션 버튼 | 待機中: `呼び出す`(주 버튼) / `取消`(텍스트 버튼)<br>呼び出し済み: `確定`(주 버튼, 초록) / `再呼出` / `取消` |

### 4-2. 카드 구성 (本日の履歴)

액션 버튼 없음. 상태 배지(`確定`/`キャンセル`)와 처리 시각만 표시. 클릭 비활성.

### 4-3. 상태 색상

| 상태 | 색상 | 배지 텍스트 |
|------|------|-------------|
| `waiting` | `#D4AC0D` (황) | 待機中 |
| `called` | `#2E86C1` (청) | 呼び出し済み |
| `confirmed` | `#1E8449` (녹) | 確定 |
| `cancelled` | `#E74C3C` (적) | キャンセル |

### 4-4. 인라인 추가 폼

Zone B의 ＋ウェイティング追加 클릭 시 Zone C 최상단에 입력 행 펼침: 고객명 / 전화번호 / 메뉴 / 희망 스태프 / 등록 버튼. 등록 시 待機中 리스트 맨 끝에 추가되고 토스트 알림.

## 5. Zone D — サイドパネル

| 섹션 | 내용 |
|------|------|
| D-1 今日の状況 | 総ウェイティング・待機中・確定・キャンセル 4개 지표 |
| D-2 平均待ち時間 | 금일 확정 건 기준 평균 대기 분 (큰 숫자 강조) |
| D-3 凡例 | 경과시간 색상 기준 안내 (10분 미만/10~20분/20분 이상) |

## 6. JS 상태 관리

| 변수/함수 | 역할 |
|-----------|------|
| `WAITING[]` | 대기 데이터 배열 (id, rank, name, phone, svc, dur, staff, joinedAt, status) |
| `curTab` | `'waiting'\|'called'\|'history'` 현재 탭 |
| `switchTab(tab)` | 탭 전환 + 리렌더 |
| `callCustomer(id)` | waiting → called, 토스트 |
| `confirmCustomer(id)` | called → confirmed, 토스트 (실제 구현 시 07_booking_form 이동) |
| `cancelCustomer(id)` | 상태 → cancelled, 토스트 |
| `reorder(id, dir)` | 대기 순서 위/아래 이동 |
| `filterText` | 검색어, 고객명 부분 일치 필터 |
| `render()` | 탭+검색 조건에 맞춰 Zone C, Zone D 갱신 |
