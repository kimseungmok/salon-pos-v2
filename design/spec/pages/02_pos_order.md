# 02 · POS 주문 화면 정의서

> **파일**: `design/mockups/ja/02_pos_order.html`  
> **목적**: 결제 플로우의 첫 단계. 담당 스태프·고객을 지정하고 시술 메뉴를 카트에 담은 뒤 계산 화면으로 넘어감.  
> **진입 경로**: 메인 런치패드 → 会計 타일 클릭  
> **다음 화면**: 카트에 항목 있는 상태에서 「会計する」 클릭 → `03_pos_payment`  
> **레이아웃**: 12컬럼 기준 좌 8(상품 패널) / 우 4(카트 패널)

---

## 화면 구조 다이어그램

```
┌─────────────────────────────────────────────────────────────────┐
│  글로벌 탑바 (44pt)                                              │
├──────────────────────────────────────────┬──────────────────────┤
│  서브바: 고객칩 + 스태프칩 + 연동버튼     │  카트 헤더           │
├──────────────────────────────────────────┤                      │
│  카테고리 탭 (施術/カラー/…)              │  카트 아이템 목록    │
├──────────────────────────────────────────┤                      │
│                                          │  할인 입력           │
│  상품 그리드 (4컬럼)                     │  메모                │
│                                          │  합계 요약           │
│                                          │  회계버튼            │
└──────────────────────────────────────────┴──────────────────────┘
```

### E-06 · 합계 요약 (Cart Summary)

| 항목 | 값 |
|------|----|
| 타입 | 동적 렌더링 영역 (`#cartSummary`) |
| 업데이트 트리거 | 카트 변경, 할인 입력 시 `renderSummary()` 호출 |

표시 행:

| # | 항목 | 조건 | 스타일 |
|---|------|------|--------|
| E-06-1 | 소계 | 항상 | `¥N,NNN` |
| E-06-2 | 할인 | `discAmt > 0` 일 때만 | `－¥N,NNN` / `#E74C3C` |
| E-06-3 | 구분선 | 항상 | `.5px solid #E5E5E0` |
| E-06-4 | 합계 | 항상 | 26px / 700 |
| E-06-5 | 소비세 | 항상 | 10px / `#999` / `税込み ¥N,NNN` |

**계산 로직**:
```
소계 = Σ(price × qty)
할인 = round(소계 × 할인율 / 100) + 할인금액
합계 = max(0, 소계 - 할인)
소비세 = round(합계 × 10 / 110)  // 세금 포함 기준
```

## G. JS 상태 관리

| 변수 | 타입 | 설명 |
|------|------|------|
| `cart` | Array | 카트 아이템 배열 |
| `nextId` | number | 카트 아이템 고유 ID 자동 증가 |
| `currentCat` | string | 현재 선택된 카테고리 |
| `PRODUCTS` | Object | 카테고리→상품 매핑 데이터 |

| 함수 | 역할 |
|------|------|
| `renderProducts(cat)` | 카테고리별 상품 그리드 렌더링 |
| `addToCart(product)` | 카트 추가 또는 수량 +1 |
| `removeItem(id)` | 카트에서 항목 제거 |
| `changeQty(id, delta)` | 수량 변경 (최소 1) |
| `clearCart()` | 카트 전체 초기화 |
| `calcTotal()` | 소계·할인·합계·세금 계산 |
| `renderCart()` | 카트 아이템 목록 DOM 렌더링 |
| `renderSummary()` | 합계 요약 DOM 렌더링 |
| `renderAll()` | 위 세 함수 일괄 호출 |
| `selectCat(el, cat)` | 카테고리 탭 전환 |
| `toggleCustomer()` | (데모) 고객 순환 전환 |
