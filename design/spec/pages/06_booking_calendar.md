# 06 · 예약 캘린더 화면 정의서

> **파일**: `design/mockups/ja/06_booking_calendar.html`  
> **목적**: 예약 관리의 중심 화면. 일/주/월/일람 4가지 뷰로 예약 현황을 시각화하고, 예약 추가·수정·노쇼 처리·대기 관리를 수행함.  
> **진입 경로**: 메인 런치패드 → 予約管理 타일 / 탑바 → 予約 탭  
> **다음 화면**: 예약 추가 → `07_booking_form` / 예약 클릭 → `07_booking_form` (수정) / 결제 이동 → `02_pos_order`  
> **레이아웃**: 캘린더 영역(flex:1) + 사이드 패널(220px, 토글 가능)

---

### B-04 · 예약 블록 (Booking Block)

시간축에 절대 위치로 배치되는 핵심 UI 요소.

| 항목 | 값 |
|------|----|
| 타입 | `position:absolute` div `.bk` |
| left/right | `3px` (컬럼 내 좌우 여백) |
| top | `toTop(startH, startM)` = `(startH-9)×40 + startM×(40/60)` px |
| height | `max(22, dur×(40/60)-2)` px |
| 보더 레이디어스 | 6px |
| 배경색 | `STATUS_COLOR[status]` |
| cursor | `pointer` |
| hover | `opacity: .85`, `box-shadow: 0 2px 8px rgba(0,0,0,.15)` |
| 블록타임 추가 스타일 | `bk-block`: 대각선 줄무늬 오버레이 |

**내부 구조 (높이에 따라 조건부)**:

| 요소 | 최소 높이 | 스타일 |
|------|-----------|--------|
| 고객명 | 항상 | 11px / 500 / white, 말줄임 |
| 시술명 | 30px 이상 | 10px / `rgba(255,255,255,.8)`, 말줄임 |
| 시간 | 50px 이상 | 9px / `rgba(255,255,255,.65)` `HH:MM (N分)` |

**파라미터 (JS BOOKINGS 배열의 각 항목)**:

```js
{
  staff: 0,              // 스태프 인덱스 (STAFF 배열 기준)
  startH: 9,             // 시작 시(hour)
  startM: 0,             // 시작 분(minute)
  dur: 60,               // 소요 시간(분)
  name: '田中 美咲',     // 고객명
  service: 'カット + カラー', // 시술명
  status: 'done'         // 상태: done/checkin/confirmed/pending/noshow/block
}
```

**STATUS_COLOR 매핑**:

| 상태 | 색상 |
|------|------|
| done | `#888` |
| checkin | `#2E86C1` |
| confirmed | `#1E8449` |
| pending | `#D4AC0D` |
| noshow | `#8E44AD` |
| block | `#555` |

**클릭 동작**: `showBookingDetail(b)` → 예약 상세 팝업 (추후 구현, 현재 alert)

## G. JS 데이터 구조

### STAFF 배열

```js
[
  {name:'Yuki', color:'#8E44AD'},
  {name:'Mika', color:'#2E86C1'},
  {name:'Kenji', color:'#E67E22'},
  {name:'Hana', color:'#888', off:true},
]
```

### BOOKINGS 배열 (14개)

각 항목의 `staff` 필드는 STAFF 배열의 인덱스(0~3).

| 함수 | 역할 |
|------|------|
| `buildDayView()` | 일 뷰 시간 라인 + 예약 블록 생성 |
| `buildWeekView()` | 주 뷰 헤더 + 7일 그리드 생성 |
| `buildMonthView()` | 42셀 달력 생성 |
| `buildListView()` | 예약 리스트 정렬·렌더링 |
| `buildMiniCal()` | 사이드 미니 달력 생성 |
| `toTop(h, m)` | 시각 → px 변환: `(h-9)×40 + m×(40/60)` |
| `switchView(v, el)` | 뷰 전환 |
| `toggleFilter(el)` | 스태프 필터 토글 |
| `toggleSide()` | 사이드 패널 토글 |
| `navigate(dir)` | 날짜 이동 |
| `showBookingDetail(b)` | 예약 상세 (추후 팝업 구현) |
