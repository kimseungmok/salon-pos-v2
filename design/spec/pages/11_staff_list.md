# 11 · スタッフ一覧 — 화면 정의서

> **파일**: `design/mockups/v2/ja/11_staff_list.html`
> **목적**: 전체 스태프 검색·직급 필터 + 신규 스태프 등록 진입점.
> **진입**: 사이드바 → スタッフ
> **이동**: 카드 클릭 → `12_staff_detail` / Zone B 「勤怠ログ（全店舗）」 → `13_staff_shift`
> **컨셉**: 09_customer_list와 동일한 마스터+사이드 패턴. 계정/권한 제어는 통근관리사이트로 이관됨 — [[12_staff_detail]] 참조.

---

## 1. 구역 구성

| 구역 | 역할 |
|------|------|
| Zone B | 직급 탭(すべて/オーナー/店長/主任/スタッフ) + 검색 + 「勤怠ログ（全店舗）」 + ＋スタッフ追加 |
| Zone C | 스태프 카드 리스트: 아바타·이름·직급배지·소속점·이번달매출/건수·재실여부 |
| Zone D | 총 스태프수/본일 재실/타점 헬프중 통계, 권한 레벨 범례 |

## 2. 권한 레벨(4단계)

| 레벨 | 색상 | 권한 |
|------|------|------|
| オーナー | `#C0392B` | 전권한 |
| 店長 | `#8E44AD` | 개점·폐점·스태프관리 |
| 主任 | `#2E86C1` | 환불/취소 승인까지 |
| スタッフ | `#117A65` | POS·예약 기본만 |

상세 매트릭스는 `12_staff_detail.md`/`33_staff_accounts.md` 참조.
