# 04 · 결제 완료 화면 정의서

> **파일**: `design/mockups/ja/04_pos_complete.html`  
> **목적**: 결제 플로우 마지막 단계. 완료 사실을 확인하고, 영수증을 출력/전송하며, 다음 예약 제안 등 후속 액션을 처리함.  
> **진입 경로**: `03_pos_payment` → 「会計を実行」 버튼  
> **다음 화면**: 새 주문 → `02_pos_order` / 예약 → `06_booking_calendar` / 고객 상세 → `10_customer_detail`  
> **레이아웃**: 좌(280px 영수증) / 중(flex:1 완료+고객) / 우(220px 다음액션)

---

## 화면 구조 다이어그램

```
┌──────────────────────────────────────────────────────────────────┐
│  글로벌 탑바                                                      │

│  영수증 뷰       │  ✓ 성공 배너                 │  다음 액션     │

│                  │                              │                │

```
