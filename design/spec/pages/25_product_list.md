# 25 · 商品一覧 — 화면 정의서

> **파일**: `design/mockups/v2/ja/25_product_list.html`
> **목적**: 카테고리별 상품(소모품/판매상품) 조회·검색, 재고상태 한눈에 확인, 등록 진입점.
> **벤치마킹**: **토스포스 100% 기반** — `salon_pos/doc/benchmarking/01_tosspos_analysis.md` §3-2(상품 대량등록) §3-3(카테고리·옵션·상품 설정) §4-6(재고관리) 그대로 구현. 프레샤 요소 가미 없음.
> **진입**: 사이드바 → 商品 · 在庫管理
> **이동**: 카드 클릭 / ＋商品追加 → `26_product_form`, 並び順を編集 → `27_product_layout`, 在庫 링크 → `14_inventory_list`/`15_inventory_history`

---

## 1. 구역 구성

| 구역 | 역할 |
|------|------|
| Zone B | 카테고리 탭(가로 스크롤) + 검색 + 商品 一括編集(엑셀 다운로드→업로드 안내, §3-2) + ＋商品追加 |
| Zone B2 | 카테고리 삭제 시 미분류 이동 안내 힌트 + 商品の並び順を編集 링크(→27) |
| Zone C | 상품 카드 그리드: 사진·이름·카테고리·가격·재고배지(품절/부족/정상)·옵션연결 배지 |
| Zone D | 총상품수/재고부족/품절 통계, 카테고리별 상품수, 재고현황·재고변동履歴 바로가기 |

## 2. 재고 배지 로직 (토스 §4-6 그대로)

`stock<=0` → `品切れ`(빨강), `stock<=lowAt` → `残り N`(주황, 부족), 그 외 `残り N`(초록, 정상). 재고 0이 되면 자동 품절 — 판매 화면(POS)에서도 동일 로직 재사용 예정.
