# 19 · クーポン管理 — 화면 정의서

> **파일**: `design/mockups/v2/ja/19_coupon_management.html`
> **목적**: 쿠폰 발행·조회·활성화 관리.
> **벤치마킹**: **토스 베이스 + 프레샤 보조** — 토스포스 §1-3(할인: 금액/비율 선택, 자주 쓰는 할인 즐겨찾기) + §3-4/§4-5(영수증 쿠폰번호: 로또번호 방식 + 할인 비밀번호)를 메인 골격으로, 프레샤 Course 5.7(Apply Rewards at Checkout)은 별도 카드로 분리해 보조 표시만.
> **진입**: 사이드바 → マーケティング

## 1. 구역 구성

| 구역 | 토스/프레샤 | 내용 |
|------|------------|------|
| Zone B | 토스 §1-3 | すべて/有効/終了 탭 + ＋クーポン発行 |
| 쿠폰 카드 리스트 | 토스 §1-3 | 코드, 이름, 할인(¥ 또는 %), 상태(有効/準備中/終了), ON/OFF 토글 |
| レシートクーポン番号設定 | 토스 §3-4/§4-5 | ロト番号方式 토글, 割引パスワード 입력 |
| プレシャ参考機能 카드 | 프레샤(보조) | 체크아웃 시 쿠폰+포인트 동시 적용 예정 노트(보라색 카드로 시각적 구분) |

## 2. 색상 구분 규칙

이번 쿠폰/캠페인/포인트 묶음부터 **토스 베이스 콘텐츠는 화면 고유 색(오렌지 #D35400)**, **프레샤 보조 콘텐츠는 항상 보라색 카드(#F5EEF8 배경 + "プレシャ参考" 태그)**로 시각적으로 구분 — 어느 부분이 메인 골격이고 어느 부분이 보조인지 한눈에 알 수 있게 함. 20/21에도 동일 규칙 적용.
