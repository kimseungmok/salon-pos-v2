# 顧客管理 (고객관리) — 화면 정의서

> 기능 규칙은 `feature_spec.md` 참조(이 문서에서 F-CUST-xx로 인용). 레이아웃/인터랙션만 정의.

---

## 화면 09 — 顧客リスト（고객 리스트）

### Zone 구조

| Zone | 내용 |
|---|---|
| Zone A | 상단바: ☰ 메뉴, ロゴ, 우측 시각 표시 (전체 화면 공통) |
| Zone B | 그룹 탭(5개) + 검색창 + 정렬선택 + 우측 액션 버튼 |
| Zone C | 고객 카드 리스트 (세로 스크롤) |
| Zone D | 하단 "顧客グループ定義 ※トスプレイス基準" 범례 |

### Zone B 상세

- 그룹 탭 라벨(좌→우): `すべて` / `🐣初回来店` / `🟡予備常連` / `🔴常連` / `⚪休眠ぎみ`
  - 각 탭에 실시간 카운트 숫자 배지(F-CUST-01 결과 기준)
  - 활성 탭: 흰 배경 + 그림자, 비활성: 투명
- 검색 입력란: placeholder = `番号、名前、メモで検索`
  - 입력 즉시(디바운스 200ms) 필터링
- 정렬 드롭다운: 기본값 `최근방문순(recent)`, 옵션에 이름순/등록일순 등 추가 가능(상세 옵션은 구현 시 결정)
- 우측 버튼: `直接積立・チャージ`(색상: #1E3A8A) — 클릭 시 고객 미지정 상태로 포인트적립/선불권충전 진입(고객 선택 모달 선행)

### Zone C 카드 레이아웃

각 카드는 1행:

```
[그룹아이콘] [그룹라벨 | 고객이름]        마스킹전화번호
                                          총방문 N回 / 최근30日 M回
                                          선불권 ¥N,000  포인트 N P
```

- 그룹아이콘+라벨: F-CUST-01 결과의 `GROUP_ICON`/`GROUP_LABEL` 맵 사용
- 카드 클릭 → 10(顧客詳細)로 이동
- 전화번호는 마스킹 표시: `010-****-1234` 형식(가운데 4자리 마스킹)

### Zone D 범례 텍스트(고정, 변경 금지 — 토스 기준 그대로)

```
顧客グループ定義 ※トスプレイス基準
🐣 初回来店 — はじめてご来店のお客様
🟡 予備常連 — 2回以上ご来店、まだ常連ではない方
🔴 常連 — 直近90日間で7回以上ご来店の方
⚪ 休眠ぎみ — 予備常連・常連だったが45日以上来店がない方
```

## 화면 10 — 顧客詳細（고객 상세）

### Zone 구조

| Zone | 내용 |
|---|---|
| Zone B | 헤더: 戻る + 아바타 + 이름/전화 + 그룹·방문 요약 + 修正 버튼 |
| Zone B2 | 메모 입력란(F-CUST-02) — 항상 노출 |
| Zone C | ポイント / プリペイド券 카드 2칸 + クイックアクション + 統計 + グラフ |
| Zone D(탭) | 基本情報 / カルテ / 履歴 (살롱 고유 — 토스에 대응 없음, 보너스 기능 유지) |

### Zone B 상세

```
‹  [아바타]  田中 美咲（090-****-5678）          [修正]
             🔴常連 ・ 来店12回 ・ 最終来店 3日前
```

- `‹` 戻る 아이콘: 09 리스트로 복귀
- 修正 버튼: 이름/전화번호 등 기본정보 수정 모달(별도 정의 불필요 — 단순 폼)

### Zone B2 메모

```html
<input class="memo-input" placeholder="顧客についてのメモを書いてください。オーナーのみ見られます">
```

- F-CUST-02 권한 규칙 적용: 일반 스태프 로그인 시 이 Zone 자체를 숨김 처리(v3 권장 — 기존 목업은 미구현)

### Zone C 상세

```
┌─ ポイント ──────────┐  ┌─ プリペイド券 ────────┐
│ 1,280P      [-] [+] │  │ ¥0(なし)        [+]   │
└──────────────────────┘  └────────────────────────┘
```

- ポイント `[+]`/`[-]`: 클릭 시 금액 직접입력 모달(소액 가감용)
- プリペイド券 `[+]`: **F-PP(선불권) 플로우 진입점** — 클릭 시 동작은 `prepaid_pass/screen_spec.md` §화면28 참조
  - 기존 잔액이 있으면 카드에 `잔액 ¥N (N일 남음)` 형식으로 표시(F-PP-03 사용 시 실시간 갱신)

クイックアクション(기존 유지): 予約 / 会計 / メッセージ送信 / VIP指定 / ブラック指定
統計: 来店回数 / 最終来店 / 誕生日
グラフ: 방문추이(토스 근거 없는 보너스 — `card-head-hint`에 "※トスプレイス基本画面にはない補助情報" 명시 유지)

### Zone D 탭(변경 없음, 살롱 고유 가치로 유지)

基本情報 / カルテ(컬러레시피·시술기록) / 履歴 — 토스에 대응 화면 없음, 의도적으로 유지.
