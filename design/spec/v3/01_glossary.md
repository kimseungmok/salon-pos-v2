## 숫자·단위 표기 규칙

| 항목 | 규칙 | 예시 |
|---|---|---|
| 화폐 | `¥` + 천단위 콤마, 일본어 「円」 텍스트는 쓰지 않고 ¥ 기호만 사용 | ¥12,000 |
| 날짜 | 年月日 형식, 요일은 괄호 | 2026年6月23日（火） |
| 시각 | 24시간 표기 | 14:30 |
| 방문횟수 | 件 또는 回 (예약/주문=件, 방문·시술=回) | 来店3回, 注文5件 |
| 인원 | 名 | スタッフ6名 |
| 시술시간 | 分 | 90分 |
| 상품종류 | 種 | 全9種 |
