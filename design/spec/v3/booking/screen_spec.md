# 予約 (예약) — 화면 정의서

## 화면 06 — 予約カレンダー

### Zone 구조

| Zone | 내용 |
|---|---|
| 좌 | 월간 미니캘린더 |
| 우 | 담당자별 컬럼 + 시간축 + 예약 블록 |

### 예약 블록

```
┌─────────────┐
│ 田中様        │ ← 배경: toPastel(担当者カラー, 0.62)
│ カット+カラー  │
│ [ノーショー]  │ ← 조건부
└─────────────┘
```

`toPastel(hex, ratio)` = 담당자 컬러를 흰색과 ratio(0.62) 비율로 혼합.

### 클릭 팝업

```
[아바타] 田中 美咲          🔴常連、総12回来店、ノーショー0回
ポイント 1,280P
カット ・ 40分 ・ 担当: Yuki
2026/06/23 14:00〜15:30
メモ: 首の後ろのくせ毛注意
                              [+ 新規予約]
```

## 화면 07 — 予約登録フォーム

### Zone 구조(좌→우 2컬럼)

| Zone | 내용 |
|---|---|
| 좌 | 顧客 / メニュー / 担当スタッフ / **予約金(F-BOOK-02a)** / 追加設定 |
| 우 | 日時選択(캘린더+시간슬롯) |

### 予約金 카드

```
予約金                                    [ON/OFF 토글]
[銀行振込 ▾]   [____20,000____]
☐ 受け取りました
キャンセル案内文（任意）
[返金は24時間以内に可能です。________]
```

- 토글 OFF 시 카드 본문 숨김(`display:none`)

### 담당자 칩 상태 표시

```
[Yuki  ✓空き]  [Mika  ✓空き]  [Kenji ✕予約あり]  [Hana  休み]
```

선택된 칩: 보더+배경 하이라이트(担当者컬러).

## 화면 08 — ウェイティング管理（토스 근거 없음, 독자기능 배지 필수）

### Zone 구조

| Zone | 내용 |
|---|---|
| Zone B | 타이틀 + `※トスにない独自機能` 배지 + 탭(待機中/呼び出し済み/本日の履歴) + `＋ウェイティング追加` |
| Zone C(좌) | 대기열 리스트 |
| Zone C(우, 패널) | 今日の状況(총건수/대기중/확정/취소) + 평균대기시간 + 경과시간 범례 |

### 대기열 행

```
[▲▼] ①  森 かれん  090-1234-5678
        カット+カラー・90分 / 指定: Yuki
        10:05 受付 ・ 19分待ち              [呼び出す] [取消]
```

대기시간 색상: `10분미만=회색 / 10~20분=주황 / 20분이상=빨강`(F-BOOK-03)
