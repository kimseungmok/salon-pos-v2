## 화면 14 — 在庫現況

```
‹戻る  在庫現況 [※トスにない独自機能]  [すべて][カラー剤][パーマ剤]...  ☐不足・品切れのみ  [変動履歴を見る]
```

```
商品名              在庫数（クイック修正）    しきい値    状態
カラー剤（ブラウン系）  [-] 24 [+]              10        正常
カラー剤（アッシュ系）  [-]  6 [+]              10        不足
パーマ液2剤            [-]  0 [+]               8        品切れ
```

우측 패널:

```
在庫サマリー
総商品数      9点
在庫不足      2点
品切れ        1点

使い方
＋／－ボタンで在庫数をその場で調整できます。在庫が0になると自動的に
品切れ表示・POS側で販売不可になります。
```

> 마지막 문장("POS側で販売不可になります")은 F-INV-00 결정에 따라 **실제로는 연결하지 않으므로 v3 구현 시 삭제하거나 "향후 검토"로 수정 필요** — 기존 v2 목업 문구를 그대로 가져오면서 발생한 모순, 본 정의서 작성 중 발견.
