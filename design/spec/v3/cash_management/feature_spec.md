# レジ金管理 (시재관리) — 기능 정의서

> 대상 화면: 22(開店準備/レジ開始金), 23(レジ締め)
> 근거: `toss_benchmarking.md` §1-7, 토스플레이스 가이드 `guide/sector/service-store/payment/cash-management`

---

## F-CASH-01. 시재 기능 개요

매장의 현금 흐름을 권종(지폐/동전)별로 카운트해 파악하는 기능. 토스 원본 기능은 3가지: `돈 넣기`(개점 시작금) / `돈 빼기`(중간 입출금) / `돈통 정산`(폐점 마감).

---

## F-CASH-02. 권종별 카운트 (22/23 공통)

### 규칙
- 권종은 **지폐(枚 단위)**: ¥10,000 / ¥5,000 / ¥1,000 — **동전(個 단위)**: ¥500 / ¥100 / ¥50 / ¥10 / ¥5 / ¥1 로 구분 표시
- 각 권종 입력란 + ±버튼(살롱 고유 보너스 — 혼자 개점하는 직원의 빠른 조정을 위해 추가, 토스 원본은 텍스트 입력 + 공유 숫자패드 방식)
- 합계(総額)는 모든 권종 × 수량의 합으로 실시간 자동계산, 화면 상단에 큰 숫자로 표시

### 차이점 — 개점(22) vs 폐점(23)

| | 개점(22) | 폐점(23) |
|---|---|---|
| 비교 기준값 | 前日締め予想額(전일 마감액) | 予想締め額 = 시작금 + 현금매출 − 환불 |
| 차액 처리 | 차액 있으면 재확인 안내만 | 차액 있으면 재확인 안내 + 마감 확정 시 차액 사유 기록 권장(v3 신규 — 토스 근거 없음, 매장 운영상 필요해 추가) |
| 확정 버튼 | `開店確定` | `閉店確定`(체크리스트 F-CASH-04 완료 후 활성화) |

---

## F-CASH-03. 직원 출근 현황 (22 좌측, 살롱 고유)

- 당일 출근 직원 목록 + 체크인/체크아웃 시각, 헬프(타지점 지원) 표시
- 토스 원본에는 없는 기능이나 멀티스태프 살롱 운영에 필요해 유지(보너스 기능으로 명시)

---

## F-CASH-04. 폐점 체크리스트 (23, 살롱 고유)

- 매출 정산서 출력·확인, 시술공간 정리, 소모품 재고 확인, 출입구 시정 등 항목형 체크리스트
- 전체 체크 완료 전 `閉店確定` 버튼 비활성

---

## 예외 케이스

| 케이스 | 처리 |
|---|---|
| 총액이 0일 때 | 차액 표시 안 함("-")— 카운트를 아직 시작하지 않은 정상 상태로 간주 |
| 차액이 정확히 0 | "¥0（一致）" 긍정 표시(녹색) |
| 차액이 음수/양수 | 빨간색 강조 + "差額があります。再確認してください。" 안내 |
