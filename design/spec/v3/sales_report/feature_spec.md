## F-SALES-01. 토스 기준 핵심 화면 (필수, 売上概況 탭)

토스 실제 매출리포트는 **매우 미니멀**하다 — 차트/그래프/랭킹이 전혀 없다.

### 구성
1. 기간 토글: `일간 / 주간 / 월간`(세그먼트, 선택 시 다크필) + 날짜 네비게이션
2. 안내문: "{기간} {HH:MM} 기준의 累計데이터입니다"
3. 핵심 숫자 3개만:
   - 実売上(굵게, 크게)
   - 注文件数
   - 返品(빨강, 음수)
4. 決済手段別売上: 현금/카드/QR코드 등 아이콘+라벨+바그래프+금액
5. 하단 버튼: `決済内訳を見る`(아웃라인) / `出力する`(블루 솔리드)

### 규칙
- 매출달력(별도 화면, 본 정의서는 17에 흡수)은 **날짜 그리드 + 일별 매출 숫자만** — 날씨 아이콘·메모 기능은 토스에 없음(`toss_benchmarking.md` 1-6 정정사항 확정). 구현 시 날씨 아이콘을 넣더라도 "토스 근거 없는 보너스"로 라벨링 필수

## F-SALES-02. 보너스 영역 (토스 근거 없음, 명확히 라벨링)

다음은 토스 기본화면에 없는 정보이며, 구현 시 카드 헤더에 "※トスプレイス基本画面にはない補助情報" 같은 힌트 텍스트를 반드시 동반한다.

| 영역 | 내용 |
|---|---|
| 売上推移 | 주별 바그래프 |
| 前年同期間比較 | 전년 대비 % |
| 商品売上(메뉴랭킹) | 시술메뉴별 순위 + 전기간비 등락률 |
| 決済・取引(상세) | 결제유형별 비율, Fresha 참고 |
| 顧客分析 | F-CUST-01 그룹별 매출 집계(고객관리 영역과 연동) |
| スタッフ実績 | 담당자별 매출/가동률(F-STAFF 영역과 연동) |
| 売上カレンダー | 날짜그리드(F-SALES-01 규칙 그대로, 날씨 보너스 제외 권장) |
| 店舗比較 | 다지점 매출 비교 |

## F-SALES-03. 매출 리포트 잠금 (PIN 보호)

- 우상단 "レポート保護: ON/OFF" 토글
- ON 상태에서 진입 시 PIN(6자리) 입력 게이트
- 토스 gitbook 실 화면 대조는 미완료(`toss_benchmarking.md` §2 잔여 항목) — 사내 분석자료 기준으로만 구현되어 있어 추후 재검증 권장
