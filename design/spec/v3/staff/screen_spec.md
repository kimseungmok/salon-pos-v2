## 화면 33 — スタッフ招待（계정 초대）— 토스 그대로

### Zone 구조

| Zone | 내용 |
|---|---|
| Zone B | 타이틀 + `＋スタッフ招待` 버튼(탭 없음) |
| Zone C | 4컬럼 테이블: スタッフ名 / 携帯電話番号 / 状態 / 액션 |
| 모달 | 招待 입력(이름+연락처) |

### 테이블 행

```
Yuki     090-****-1234    連結済み      削除
Rio      090-****-4567    待機中        再送信
```

### 초대 모달

```
スタッフ招待
スタッフ名     [______]
携帯電話番号   [______]
                    [招待する]
```

하단 안내(고정 문구): `※ トスプレイス実画面の通り：招待された人にアルリムトークが送信され、招待状を受けて同意すると売上が連動されます。PIN・権限レベルの管理はPOSの範囲外（勤怠管理サイトで運用）。`
