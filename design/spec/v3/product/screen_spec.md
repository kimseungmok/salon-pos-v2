# 商品・カテゴリ — 화면 정의서

> 기능 규칙은 `feature_spec.md` 참조(F-PROD-xx로 인용).

---

## 화면 25 — 商品リスト + 등록 모달

### Zone 구조

| Zone | 내용 |
|---|---|
| Zone B | 서브탭(商品/オプション/カテゴリ/割引) + `＋商品追加` |
| Zone C | 상품 카드 리스트(클릭 시 모달로 편집 진입, 페이지 이동 없음) |
| 모달 | F-PROD-01 입력 항목 |

### 등록 모달

```
商品追加・編集
[스와치 11색 그리드]          選択中: ■(紫)
キオスク露出           [ON/OFF]
商品名     [______________]
カテゴリ   [カット] [カラー] [パーマ] [+新規カテゴリ]   ← 単一選択
価格       [______] 円   □ 毎回直接入力する
オプション [長さ] [コース追加] [色の濃さ] [サイズ] [本数追加]  ← 複数選択
                                  [キャンセル]  [保存する]
```

## 화면 26 — 商品登録（→25へ統合, 리다이렉트 전용）

```html
<!-- 9줄, 로직 없음 -->
location.replace('25_product_list.html' + (id ? '?openId='+id : '?open=1'));
```

`?openId=N` 쿼리로 25 모달을 편집 모드로 자동 오픈(하위호환 유지).
