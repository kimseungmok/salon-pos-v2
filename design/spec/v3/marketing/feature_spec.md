## F-MKT-01. 시즌널 쿠폰 발행 (19)

### 정의
매장에 방문했던(마케팅 수신 동의한) 고객에게 발송하는 1회성 쿠폰.

### 3-Step 생성 흐름
1. **시즌 템플릿 선택**(아이콘 그리드): ホワイトデー/お花見/卒業/雪の日/バレンタイン/お誕生日/クリスマス/お正月/こどもの日/親の日/試験期間/ハロウィン/雨の日/真夏日 — **현재 제공되는 기본 템플릿 내에서만** 선택 가능(자유 텍스트 템플릿 생성 불가)
2. **혜택 유형**: `割引`(금액 또는 %, 적용범위[총주문금액/특정상품]+최소주문금액) 또는 `プレゼント`(상품 지정, 복수 선택 시 가장 비싼 상품 1개만 무료)
3. **유효기간**: 7日/14日/30日/常時

### 규칙
- 마케팅/광고 수신 동의하지 않은 고객에게는 발송 불가
- 발행 즉시 활성(有効) 상태, 만료일 도달 시 자동 終了

### 영수증 쿠폰번호(별개 기능, 19에 병존)
- ロト番号方式: 영수증에 추첨번호 인쇄, 방문 시 제시하면 할인 적용
- 割引パスワード: 영수증 하단 인쇄 암구호, POS 입력 시 할인 적용

## F-MKT-02. 캠페인 — 범위 확정 (토스 근거 없음)

토스에는 **상시 캠페인 관리 화면이 없다**(`toss_benchmarking.md` §1-10). 마케팅 기능은 ①즉석 할인(F-PAY 영역의 카트 내 "割引" 버튼, 자주쓰는 할인 등록·자동할인 설정 가능하지만 상시 대시보드 아님) ②F-MKT-01 쿠폰 ③F-MKT-03 포인트 3가지로 완결된다.

→ **20(キャンペーン管理)은 토스에 대응 기능 없는 살롱 고유 자산**으로 명시 유지(평일 나이트타임 할인, 생일월 캠페인, 추천 프로그램 등 — Fresha 참고 보너스 포함). 화면에 `※トスにない独自機能` 배지 필수.

## F-MKT-03. 포인트 정책 (21)

### 토스 기준 4항목 (이 4개가 전부)
1. **付与率(%)**
2. **最低利用ポイント(P)**
3. **付与対象商品**: 全商品 / 一部除外
4. **ポイント利用対象商品**: 全商品 / 一部除外

전체 ON/OFF 토글(회계화면에서 포인트 결제 허용 여부)과 함께 운용.

### 보너스 영역(토스 근거 없음, 명확히 라벨링)
- 1포인트의 가치(¥ 환산), 포인트 유효기간(일) — "プレシャ参考"로 분리 표기
- 로열티 티어(ブロンズ/シルバー/ゴールド/プラチナ) — Fresha 참고, 토스에 없음
