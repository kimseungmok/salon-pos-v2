# 기능 전체 명세

## 1. 결제

### 주문 페이지
- 상품 패널: 카테고리 탭 전환, 그리드 표시, 위치 지정, 표시/비표시, 북마크, 페이지 추가/삭제, 상품 추가 폼 연동
- 카트 패널: 추가, 수량 변경, 삭제, 초기화, 할인(% / 금액), 쿠폰 적용, 메모
- 고객 연결: 검색, 선택, 포인트 잔액 표시, 담당 스태프 지정, 예약 연동
- 이동: 계산 페이지, 예약 화면, 고객 상세

### 계산 페이지
- 금액: 주문 요약, 소계/할인/합계, 포인트 사용, 쿠폰 코드, 상품권/선불권
- 결제수단: 현금(거스름돈), 카드, QR/PayPay/LINE Pay, 분할결제, 후불, 외상
- 기타: 팁 입력, 현금영수증, 결제 실행

### 결제 완료 페이지
- 완료 정보: 결제 요약, 포인트 적립, 거스름돈, 커미션 반영
- 영수증: 프린터, PDF, LINE/SMS 전송
- 다음 액션: 초기화, 고객 상세, 다음 예약

### 거래 이력 페이지
- 조회: 날짜/수단/고객 필터, 상세 패널, CSV
- 취소/환불: 전체 취소, 부분 환불, 사유 기록, 포인트 복원

## 2. 예약

### 예약 캘린더 페이지
- 뷰: 日/週/月/一覧, 날짜 네비, 오늘 버튼
- 필터: 담당 스태프, 상태, 룸/좌석
- 예약 블록: 상세 팝업, 드래그 리스케줄, 담당자 변경, 결제 이동, 노쇼 처리
- 추가: 예약 추가, 블록타임, 대기 목록, 통계 패널

### 예약 추가/수정 페이지
- 기본: 고객, 날짜/시간, 담당, 룸/좌석, 메뉴(복수), 소요시간 자동
- 옵션: 반복 예약, 승인 여부, 메모, 알림톡 발송

### 노쇼/대기 관리 페이지
- 노쇼: 목록, 횟수 누적, 블랙리스트 기준, 알림
- 대기: 목록, 취소 시 자동 알림, 예약 전환, 순번

## 3. 고객

### 고객 목록 페이지
- 검색: 이름/전화, 태그 필터, VIP/휴면/블랙 탭, 생일 필터, 방문빈도 정렬, CSV

### 고객 상세 페이지 (탭)
- 기본 정보: 수정, 월별 그래프, 포인트, 마지막 방문, 소개인
- 카르테: 시술 기록, 컬러 레시피, 복제, 사진, 태그/메모
- 이력: 시술/구매/예약/포인트/외상
- 액션: 예약, 결제, 메시지 발송, VIP/블랙 지정

## 4. 직원

### 직원 목록 페이지
- 카드 그리드, 역할 필터, 이달 매출/건수, 직원 추가

### 직원 상세 페이지
- 기본: 이름/역할/색상/사진/연락처/입사일
- 급여: 커미션 비율, 인센티브 기준, 급여 계산
- 근무: 출퇴근, 시프트, 휴가/잔업, 목표/평가
- 매출: 이달 합계/건수/커미션, 월별 그래프

### 시프트/스케줄 페이지
- 주간 뷰, 등록/수정, 휴무일, 반복
- 출퇴근: 버튼, 실제 시간 집계, 지각/조퇴, 월별 요약

## 5. 재고

### 재고 현황 페이지
- 카테고리 탭, 카드(재고량), 부족 경고, 검색, CSV
- 입고/출고/폐기/조정/이력
- 발주: 공급업체, 발주서, 이력, 자동 기준

## 6. 매출

### 대시보드 페이지
- KPI: 오늘/이달 매출, 객단가, 재방문율, 신규비율, 포인트
- 차트: 일/주/월/분기/연, 추이, 결제수단, 시간대, 직원별, 상품별
- 내보내기: CSV, PDF

### 거래 이력 페이지 (결제 탭과 공유)

## 7. 마케팅

### 쿠폰/캠페인 페이지
- 쿠폰: 목록, 생성(% /금액/무료), 유효기간, 횟수 제한, 코드 발행, 이력
- 캠페인: 목록, 대상 고객, 발송 일정, 결과 통계
- 발송: LINE, SMS, 생일 자동, PUSH, 템플릿
- 포인트/추천: 적립/사용 정책, 추천인 제도, 이벤트

## 8. 매장

### 개점/마감 페이지
- 개점: 시재 입력(권종별), 전일 현황, 오늘 예약 미리보기, 시작 버튼
- 마감: 매출 요약, 수단별 집계, 현금 차액, 직원별/커미션/팁, PDF, 완료

### 좌석/룸 관리 페이지
- 목록, 추가/수정/삭제, 사용 여부, 예약 연동

## 9. 시스템

### 설정 페이지
- 연동: LINE, SMS, 카드단말기, 프린터, 바코드
- 데이터: 백업/복원, 온/오프라인, 감사로그, 접속기록
- 화면: 다크/라이트, 글자 크기, 버전
- 권한(관리자): 역할/권한, 메뉴 제한, API

### 로그인 페이지
- 매장 선택, 스태프 선택, PIN 입력, 권한 분기, 자동 잠금
