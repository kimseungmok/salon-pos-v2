# salon-pos-v2

일본 살롱 전용 POS 앱 — 설계 및 구현 프로젝트

## 타겟 기기

| 항목 | 내용 |
|------|------|
| 기기 | iPad Air 2 (2014) |
| 해상도 | 2048 × 1536 (Retina, 264ppi) |
| 화면 | 9.7인치, 비율 4:3 |
| OS | iPadOS 16 |
| 프레임워크 | Flutter (iOS 12+) |
| 모크업 기준 | 1024 × 768 논리 포인트 (4:3)

## 구조

```
design/
  mockups/    화면별 UI 모크업 (HTML)
  spec/       기능 명세 (Markdown)
  assets/     디자인 리소스
lib/          Flutter 소스 코드 (구현 단계에서 추가)
```

## 진행 순서

1. 기능 명세 확정 → `design/spec/`
2. 화면별 모크업 → `design/mockups/`
3. DB 스키마 설계
4. Flutter 구현 → `lib/`
